== Description ==

Call Now, Whatsapp, Messenger, Viber, Telegram, Email, Sms, Skype call, Skype chat, Share on social, Waze, Google maps.

very useful plugin that adds a contact bar to your website for mobile visitors the plugin will allow site owner to add up 1 tab in bottom such as phone, mail, sms, google map, waze, whatsapp, skype call chat, viber, social bar, links and more.
You can find more information on our website (<a href="https://www.mobiletabs.co/">www.mobiletabs.co</a>)

<h4>Multisite Support</h4>
Mobile Tabs does not support multisite yet.

<h4>Features</h4>

1. Fully responsive, even backend.
2. RTL and LTR layout.
3. Option to choose icon or text tab, Option to upload your own icons.
4. Option to exclude pages.
5. drag and drop order of icons.
6. Analystics, events integrate on plugin to track on real time.
7. Share tab for main social media, sms, mail, whatsapp and more.
8. Z-index option to prevent overlaping. 

== Installation ==

1. From your WordPress Dashboard go to 'Plugins' > 'Add new' and search for 'call now mobile tabs'.
2. Click 'Install Now' (the blue button)
3. Click activate
4. Go to 'Settings' > 'mobile tabs' and strat using it.

OR

1. Upload "Mobile Tabs" to the "/wp-content/plugins/" directory
2. Activate the plugin through the Plugins menu in WordPress
